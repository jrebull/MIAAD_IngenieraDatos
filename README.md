### **<font color="#003CA6">Contenido Temático</font>**

| Sem. | Tema | Descripción |
|:---:|---|---|
| 1 | Introducción a la Ingeniería de Datos | Definición, importancia en ciencia de datos, roles y responsabilidades del ingeniero de datos |
| 2–5 | Bases de Datos | BD Relacionales (SQL, normalización, índices, transacciones), BD NoSQL (columnares, documentales, clave-valor, grafos), procesamiento distribuido (Hadoop, Spark) |
| 6–10 | Recolección y Almacenamiento de Datos | ETL vs ELT, herramientas de recolección, Data Warehouses (arquitecturas y tecnologías), Data Lakes (concepto, ventajas y desafíos) |
| 10–14 | Procesamiento, Transformación y Limpieza | Apache Spark y su ecosistema, procesamiento en tiempo real vs por lotes, streaming de datos, data profiling, data cleansing, linaje de datos |
| 15–16 | Infraestructura y Despliegue | Arquitecturas escalables (on-premise vs cloud), contenedores y orquestación (Docker, Kubernetes), despliegue de modelos y pipelines de datos |
